# Charles-Hall-Portfolio

## Description
This site is the personal portfolio of Charles Hall. It showcases his previous web development projects, as well as some personal information, a link to his resume, and a form to contact him. It also provides access to his GitHub, LinkedIn and Facebook profiles.

## Installation
N/A

## Usage
The about me link in the navigation will direct users to a page with personal information about Charles and an avatar. The portfolio link will direct users to Charles' past works. The contact link brings up contact information and a form to email him. The resume link will bring up a link to a downloadable resume. The links in the footer bring users to Charles' profiles on their respective apps.
